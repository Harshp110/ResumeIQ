# ResumeIQ

ResumeIQ is an AI-powered resume evaluation platform that analyzes resumes
to generate ATS scores, skill-match insights, and actionable improvement
recommendations.

## Features
- ATS compatibility scoring
- Skill gap detection
- Resume quality assessment
- AI-generated improvement suggestions
- Clean analytics dashboard

## Tech Stack
- Django
- Python
- spaCy (NLP)
- HTML / CSS
- SQLite
